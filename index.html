Jsdoc2-template-bootstrap

Beautiful Template with Bootstrap for JsDoc2.

Download this project as a .zip file Download this project as a tar.gz file

Abstract

This template make beautiful and functional documents. Only useable on JsDoc2.

Features

  • Overview
  • Beautiful code preview
  • Inheritance tree
  • Functional member documentation

Install

To produce output files you must use a template to format the generated documentation. Use the @-t@ option to specify which template you wish to use.

java -jar jsrun.jar app/run.js myscripts/ -t=templates/jsdoc

Show the official documents, if you want more informations.
